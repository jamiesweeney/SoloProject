Overview

Welcome the the peoplecount.cf!

This site provides a front-end for users to interact with my room occupancy prediction system designed and implemented for my level 4 solo project.

Smart sensor boxes (based on Raspberry Pis) communicate information on their surroundings to this site, which uses machine learning to make predictions on the occupancy of these areas. The predictions produced are available through a HTTP API, which this site and potentially others can use to serve occupancy predictions to interested parties.


Building and prediction information

From the home page, you can see all the available buildings, clicking on one will take you to a floor-by-floor breakdown of that building. The colour of each floor represents the total average occupancy per room, with a total amount also being given.

You can also see a room-by-room breakdown by clicking on a specific floor. The colour of each room represents the total occupancy per room, with a number also being given.

Clicking on a room will take you to a page for that room that shows:

  • The room name/description.
  • The current prediction and the time created.
  • All predictions for this room over the past 24 hours.


Users

Most people using this system will not require a login, as the public API endpoints are exposed without authentication requirements. Admins, however, will be given a username and password that will grant access to the admin panel and any admin API endpoints. To log in and logout, user the navigation buttons found in any page header.

Once logged in, visit the admin panel and read the help section to familiarise yourself with the admin tools.


The source code and documentation for the entire project, including smart sensor box functionality will be available here after project completion.
